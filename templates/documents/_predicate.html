Predicate Functions

and
Performance: O(n)
Arguments: [optional] {boolean} arguments
Description: Checks that all provided values are truthy

Example

    
        j.and();

        //true

        j.and(true, 1);

        //true

        j.and(true, false);

        //false
isArray
Performance: O(1)
Arguments: {any} value
Description: Returns true if value is an array, otherwise false

Example

    
        j.isArray([]);

        //true

        j.isArray({});

        //false

or
Performance: O(n)
Arguments: [optional] {boolean} arguments
Description: Checks that any of the provided values are truthy

Example

    
        j.or();

        //false

        j.or(true);

        //true

        j.or(true, false, 'true');

        //true

        j.or(0, false, null);

        //false
